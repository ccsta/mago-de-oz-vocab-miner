# 🎸 Mägo de Oz Vocab Miner

Este projeto surgiu da necessidade de automatizar a extração de vocabulário relevante para o estudo da língua espanhola, 
utilizando como base as letras da banda Mägo de Oz.

## 🚀 Funcionalidades
- **Sanitização de Dados:** Limpeza automática de pontuação específica do espanhol (¡, ¿, etc.).
- **Filtragem de Stopwords:** Ignora artigos e conjunções comuns para focar em verbos e substantivos.
- **Análise de Frequência:** Gera um ranking das palavras mais utilizadas na música informada.

## 🛠️ Tecnologias Utilizadas
- **Python 3**
- **Biblioteca `re`** (Expressões Regulares)
- **Biblioteca `collections`** (Counter para análise de dados)
  
## 📖 Como usar
1. Cole a letra da música na variável `letra`.
2. Execute o script:
   ```bash
   python minerador_mago.py
